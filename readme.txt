== Installation ==

This section describes how to install the plugin and get it working.

= Automatic installation =

1. Go to Plugins > Add New > Upload and select the .zip file from your hard disk
2. Click the "Install now" button
3. Activate the plugin through the 'Plugins' menu in WordPress

= Manual installation =
1. Download a copy (zip file) of this plugin from WordPress.org.
2. Unzip the zip file.
3. Upload the unzipped directory to your website's plugin directory (`/wp-content/plugins/`).
4. Log into WordPress Dashboard of your website.
5. Activate this plugin through the "`Plugins`" tab.

After installation, a "`Snowfall Options`"  menu item will appear in the "`Appearance`" section. Click on this in order to view plugin's administration page.

= Can I use this plugin on my language? =
Yes. But If your language is not available then you can make one. This plugin is ready for translation. The `sublimejsedit.pot` file is included and placed in "`languages`" folder. You need to just translate asiancoderswcs_text.pot file.
